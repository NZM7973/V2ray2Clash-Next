# V2ray2Clash Next

V2Ray 订阅转 Clash 订阅链接/配置文件工具，支持 **流量显示**。

## ✨ 特性

- **协议支持**: VLESS, VMess, Hysteria2, TUIC
- **流量显示**: 在 Clash 中显示 XUI 面板的流量使用情况（需源订阅支持）
- **多模式**: 支持生成配置文件或订阅链接
- **一键启动**: 提供 Bat 脚本快速使用

##  使用方法

1. 双击运行 `启动.bat`
2. 粘贴你的 V2Ray 订阅链接
3. 选择输出模式:
   - **模式 1 **: 直接转化成clash订阅链接（仅能在本地使用，无法给别的设备使用
   - **模式 2**: 直接生成 `config.yaml` 到 `订阅文件` 目录，可拖入 Clash 使用 (无流量显示)。
   - **模式 3**: 同时执行以上两种操作。
