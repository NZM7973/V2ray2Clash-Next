# V2ray2Clash Next

V2Ray 订阅转 Clash 配置文件工具，支持 **流量显示**。

## ✨ 特性

- **协议支持**: VLESS, VMess, Hysteria2, TUIC
- **流量显示**: 在 Clash 中显示 XUI 面板的流量使用情况（需源订阅支持）
- **多模式**: 支持生成本地配置文件或订阅链接
- **一键启动**: 提供 Bat 脚本快速使用

## 📦 安装教程

如果你是第一次使用，请按照以下步骤操作：

1. **下载项目**
   - 点击右上角绿色的 `Code` 按钮 -> `Download ZIP`。
   - 解压下载的压缩包到任意文件夹。

2. **安装 Python 环境**
   - 访问 [Python 官网](https://www.python.org/downloads/) 下载并安装最新版 Python。
   - **重要**：安装时务必勾选 `Add Python to PATH` (将 Python 添加到环境变量)。

3. **安装依赖**
   - 在项目文件夹空白处右键 -> `在终端打开` (或 `Open in Terminal`)。
   - 输入以下命令并回车：
     ```bash
     pip install -r requirements.txt
     ```

## 🚀 使用方法

1. 双击运行 `启动.bat`
2. 粘贴你的 V2Ray 订阅链接
3. 选择输出模式:
   - **模式 1 (推荐)**: 直接转化成clash订阅链接
   - **模式 2**: 直接生成 `config.yaml` 到 `订阅文件` 目录，可拖入 Clash 使用 (无流量显示)。
   - **模式 3**: 同时执行以上两种操作。
